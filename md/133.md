#### 133 获得某天后的1~n天

```python
import calendar
from datetime import date,datetime

def getEverydaySince(year,month,day,n=10):
    i = 0 
    while i < n:
        for d in calendar.Calendar().itermonthdates(year,month):
            if i >= n:
                break
            if d < date(year,month,day): 
                continue 
            yield d
            i += 1
```
